# SkillSync: Volunteer & NGO Connector

SkillSync is a MERN-stack platform designed to bridge the gap between non-profits and skilled professionals. NGOs can list specific skill requirements, and volunteers can find opportunities that match their expertise.

## 🚀 Features
- **Dual Dashboards:** Separate interfaces for Volunteers and NGO administrators.
- **Secure Auth:** JWT-based authentication with protected routes.
- **Skill Matching:** Filter-based matching system (AI integration pending).
- **Responsive Design:** Built with React and Tailwind CSS.

## 🛠 Tech Stack
- **Frontend:** React.js, Tailwind CSS, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (via MongoDB Compass/Atlas)
- **Auth:** JSON Web Tokens (JWT)

## 📋 Prerequisites
- Node.js installed
- MongoDB Compass running locally or a MongoDB Atlas URI
